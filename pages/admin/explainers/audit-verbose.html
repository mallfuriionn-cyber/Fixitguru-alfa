Verbose System Logging

Tento režim slouží k hloubkovému ladění (debuggingu) Jádra Synthesis. Pokud je aktivní, systém zaznamenává:

  • Surové neuronové prompty odesílané do Jádra.
  • Odpovědi API včetně metadat o latenci.
  • Každý pohyb na Event Busu mezi moduly.

⚠️ POZOR: Zapnutí tohoto režimu může způsobit rychlé zaplnění localStorage prohlížeče (Matrix DB Limit).
